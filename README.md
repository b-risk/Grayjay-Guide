# Grayjay Guide

This is a comprehensive guide to getting the best experience with Grayjay, which covers topics from client usage to YouTube blocks workarounds, and more to be planned.

## Introduction to Grayjay

To start, we need to understand that Grayjay is *not a YouTube client,* Grayjay is an aggregated media client for following and supporting creators directly, rather than the platform. YouTube has a history of blocking third party clients, and even so the developers have done an excellant job supporting the platform, but we still cannot rely on just one platform for following creators. If your creator provides content elsewhere, follow them there, if it's a paid platform like Patreon then you should subscribe to them there. We should get used to following and supporting creators on multiple platforms, and in this guide I hope to make the transition as easy as possible.
